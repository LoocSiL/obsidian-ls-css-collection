## Описание

Этот CSS-сниппет предназначен для изменения цвета **текста названий файлов** в проводнике Obsidian (File Explorer) в зависимости от их типа (расширения). Он помогает визуально различать заметки, вложения и другие файлы, что ускоряет навигацию по хранилищу.

Сниппет интегрируется с плагином Style Settings, предоставляя удобный интерфейс для настройки цветов и включения/отключения раскраски для каждой категории файлов.

## Основные возможности

- **Категоризация файлов:** Все файлы разделены на логические группы для удобной настройки.
- **Раскраска текста:** Изменяется цвет названия файла, оставляя иконки и фон без изменений (для совместимости с другими сниппетами).
- **Гибкое включение/отключение:** Можно независимо включать или выключать раскраску для каждой категории (Markdown, PDF, изображения, код и т.д.).
- **Настройка цветов:** Для каждой категории можно задать свой цвет через интерфейс Style Settings.
- **Управление активным файлом:** Есть опция, позволяющая либо сохранять цвет файла, когда он открыт (активен), либо возвращать ему стандартный цвет темы.
- **Широкая поддержка расширений:** Сниппет охватывает большинство популярных типов файлов, используемых в Obsidian и за его пределами.

## Установка

1.  Сохраните файл сниппета (`LS_File Explorer - File Type Colors.css`) в папку `.obsidian/snippets/` вашего хранилища Obsidian.
2.  В Obsidian откройте **Настройки** (Settings).
3.  Перейдите в раздел **Внешний вид** (Appearance).
4.  Прокрутите вниз до блока **CSS-сниппеты** (CSS snippets).
5.  Нажмите кнопку **Обновить** (Reload), чтобы увидеть список доступных сниппетов.
6.  Включите сниппет `LS File Explorer - File Type Colors` с помощью переключателя.

**Важно:** Для доступа к настройкам цветов убедитесь, что у вас установлен и включен плагин **Style Settings**. После включения сниппета в настройках Style Settings появится новый блок `💢 LS CSS File Explorer - File Type Colors`.

## Настройка внешнего вида

Все настройки производятся через плагин **Style Settings**. Найдите соответствующий блок в списке его настроек.

![Настройка](/images/FileTypeColors_00.png)

### 1. Общие настройки

- **Применять цвет для активного файла:**
    - **Включено:** Если файл открыт в активной вкладке, цвет его названия в проводнике будет соответствовать типу файла (согласно настройкам ниже).
    - **Выключено (по умолчанию):** Активный файл будет отображаться стандартным цветом, который использует ваша тема для активных элементов (обычно это более яркий или контрастный цвет). Цвета типа файла при этом игнорируются.

![Общие настройки](/images/FileTypeColors_01.png)

### 2. Файлы Obsidian

В этой секции настраиваются основные типы файлов, используемые непосредственно в Obsidian.

- **Markdown:** Основные заметки (расширение `.md`). Исключает файлы Excalidraw, которые имеют двойное расширение `.excalidraw.md`.
- **Canvas:** Файлы для работы с Canvas (`.canvas`).
- **Excalidraw:** Рисунки и диаграммы, созданные в плагине Excalidraw (`.excalidraw` и `.excalidraw.md`).
- **Base:** Файлы с расширением `.base` (используются некоторыми плагинами).

Для каждой из этих категорий есть переключатель **Включить** и поле выбора **Цвет**.

![Файлы Obsidian](/images/FileTypeColors_02.png)

### 3. Вложения

Эта секция отвечает за медиафайлы и документы, которые часто хранятся в хранилище Obsidian.

- **PDF:** Документы в формате PDF (`.pdf`).
- **Изображения:** Растровые и векторные изображения (`.png`, `.jpg`, `.gif`, `.svg`, `.webp` и др.).
- **Видео:** Видеофайлы (`.mp4`, `.mkv`, `.mov`, `.webm` и др.).
- **Аудио:** Аудиофайлы (`.mp3`, `.wav`, `.flac`, `.ogg` и др.).

Для каждой категории есть переключатель **Включить** и поле выбора **Цвет**.

![Вложения](/images/FileTypeColors_03.png)

### 4. Внешние файлы

Файлы, которые не являются нативными для Obsidian, но могут быть добавлены в хранилище.

- **MS Office:** Документы Microsoft Office (`.docx`, `.xlsx`, `.pptx`, а также `.csv`, `.rtf` и устаревшие форматы `.doc`, `.xls`, `.ppt`).
- **Код:** Файлы с исходным кодом и данными (`.py`, `.js`, `.css`, `.html`, `.json`, `.yaml`, `.xml`, `.sh`, `.sql` и др.).
- **Разное:** Категория для всех остальных типов файлов, которые не попали в предыдущие группы. Цвет из этой категории применяется к ним, если она включена.

Для каждой категории есть переключатель **Включить** и поле выбора **Цвет**.

![Внешние файлы](/images/FileTypeColors_04.png)

## Примеры использования

- **Визуальная типизация:** Сделайте все Markdown-файлы голубыми, а Canvas — розовыми, чтобы быстро находить нужный тип контента.
- **Выделение вложений:** Настройте для изображений и PDF яркие цвета (например, желтый и красный), чтобы они сразу бросались в глаза в списке заметок.
- **Разграничение проектов:** Если вы используете Obsidian для заметок по программированию, можно выделить файлы с кодом отдельным приглушенным цветом.
- **Совместное использование с Background Pills:** Этот сниппет отлично работает вместе со сниппетом "LS File Explorer - Background Pills", раскрашивая текст внутри цветных плашек, создавая более информативный интерфейс.

## Совместимость

Сниппет изменяет только цвет текста и не затрагивает структуру проводника. Он должен быть совместим с большинством других сниппетов и тем оформления, включая "LS File Explorer - Background Pills". В случае конфликтов убедитесь, что этот сниппет загружается после других, чтобы его правила имели приоритет (порядок в списке сниппетов можно менять перетаскиванием).
